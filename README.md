🍔 Food Delivery Web Application

A full-stack Food Delivery Web Application built using the MERN Stack. Users can browse restaurants, view food items, add items to the cart, place orders, and make online payments.

🚀 Features

- User Registration & Login
- Browse Restaurants
- View Food Menu
- Add and Remove Items from Cart
- Place Food Orders
- Online Payment Integration
- Order History
- Responsive User Interface

🛠️ Technologies Used

Frontend
- React.js
- Vite
- Redux Toolkit
- CSS

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
